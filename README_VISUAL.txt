🎯 ¿QUÉ HE CREADO?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✅ 8 PÁGINAS PRINCIPALES
     • Home (/)              - Catálogo con filtros
     • Login (/login)        - Iniciar sesión
     • Registro (/registro)  - Crear cuenta
     • Detalle (/producto)   - Página de producto
     • Carrito (/carrito)    - Carrito y checkout
     • Perfil (/perfil)      - Perfil de usuario
     • Pedidos (/mis-pedidos)- Historial de pedidos
     • Admin (/admin)        - Panel de administración

  ✅ FUNCIONALIDADES PRINCIPALES
     • Autenticación con JWT
     • Catálogo de productos
     • Carrito persistente en localStorage
     • Sistema de pedidos
     • Panel de administración (CRUD)
     • Perfil de usuario
     • Diseño 100% responsive

  ✅ TECNOLOGÍA
     • React 19.2.4
     • Vite (muy rápido)
     • Tailwind CSS (hermoso)
     • React Router (navegación)
     • JavaScript ES6+

🚀 CÓMO EMPEZAR (5 MINUTOS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. INSTALAR DEPENDENCIAS
     $ npm install

  2. VERIFICAR .env
     VITE_API_URL=http://localhost:3000/api

  3. EJECUTAR FRONTEND
     $ npm run dev
     📱 Abre: http://localhost:5173

  4. EN OTRA TERMINAL, EJECUTAR BACKEND
     $ cd ../CanoJoel_API
     $ npm install
     $ npm run dev
     🌐 Debe estar en: http://localhost:3000

  ✅ ¡LISTO! Puedes usar la aplicación
